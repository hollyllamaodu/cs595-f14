              <h1 id="page-title">JSF 2.3 has been approved</h1>
                          </div>
          
          
          <div id="content">
          <div class="article">

    <div id="blog-submitted">Posted by <a href="/blog/mriem">mriem</a> on September 23, 2014 at 12:08 PM PDT</div> 
  
    <p>The ballot for JSF 2.3 has passed and we have approval. So now it is time to move forward with EG formation and from there it is off to work! How can you stay up to date to what is happening?</p>
<p>1. Subscribe to the spec users list, see <a href="https://java.net/projects/javaserverfaces-spec-public/lists" title="https://java.net/projects/javaserverfaces-spec-public/lists">https://java.net/projects/javaserverfaces-spec-public/lists</a> and select the <a href="mailto:users@javaserverfaces-spec-public.java.net">users@javaserverfaces-spec-public.java.net</a> list<br />
2. Keep track of the issues being resolved on the issue tracker, see <a href="https://java.net/jira/browse/JAVASERVERFACES_SPEC_PUBLIC" title="https://java.net/jira/browse/JAVASERVERFACES_SPEC_PUBLIC">https://java.net/jira/browse/JAVASERVERFACES_SPEC_PUBLIC</a><br />
3. Try out our SNAPSHOT builds, see <a href="https://maven.java.net/content/repositories/snapshots/org/glassfish/javax.faces/" title="https://maven.java.net/content/repositories/snapshots/org/glassfish/javax.faces/">https://maven.java.net/content/repositories/snapshots/org/glassfish/java...</a> and look for 2.3.0-mXX-SNAPSHOT builds<br />
4. Join the Mojarra users list if you have more questions specific to trying out the 2.3 runtime</p>
<p>Enjoy!</p>
