	<h3 class="entry-title">
			Meet the Java EE 8 Experts at JavaOne
	    </h3>

    <h4 class="entry-meta">By delabassee on <a href="#">Sep 22, 2014</a></h4>

    <div class="entry-body">
                                        	<p>During JavaOne, you will have many occasions to discuss with different Java EE Specification Leads or other key Java EE staff:</p> 
  <p> </p> 
  <ul> 
    <li><a href="https://blogs.oracle.com/theaquarium/entry/javaone_java_ee_thirsty_bear">Java EE Appreciation Event</a> ... and probably other J1 Parties!</li> 
    <li><a href="https://blogs.oracle.com/theaquarium/entry/why_join_the_javaone_glassfish1">GlassFish Community Event</a></li> 
    <li><a href="https://oracleus.activeevents.com/2014/connect/sessionDetail.ww?SESSION_ID=3031">Meet the Java EE Specification Leads BoF</a></li> 
    <li><a href="https://oracleus.activeevents.com/2014/connect/sessionDetail.ww?SESSION_ID=5639">CDI 2.0 BoF</a></li> 
    <li><a href="https://oracleus.activeevents.com/2014/connect/sessionDetail.ww?SESSION_ID=3171">What Do We Want in JMS 2.1 BoF</a></li> 
    <li><a href="https://oracleus.activeevents.com/2014/connect/sessionDetail.ww?SESSION_ID=4241">What’s Next for JSF BoF</a></li> 
    <li><a href="https://oracleus.activeevents.com/2014/connect/sessionDetail.ww?SESSION_ID=3080">What’s New in JTA BoF</a> </li> 
    <li>The Java EE Pod in the <a href="https://www.oracle.com/javaone/floor-plans/index.html">Pavilion</a>'s JavaHub</li> 
    <li>And many other Java EE related sessions... &nbsp;</li> 
  </ul> 
  <p>This year, we have also arranged a <strong><em>Meet the Experts</em></strong> area. It is a designated space in the JavaHub where most of the Specification Leads will be present at a dedicated time (see schedule below).  Oracle's Java EE Evangelists and Heather VanCura from the JCP will also be present.
It will be the ideal place to carry on informal conversations about Java EE8. Also, keep in mind that the various Expert Groups are being formed so it is also the perfect time to ask what it takes to join an EG!</p>
