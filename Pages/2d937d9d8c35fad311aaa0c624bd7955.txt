<div class="entry-content"><p>by 				 <a href="http://uk.geekout.ee/speakers/simon-ritter/">Simon Ritter</a>&nbsp;
				 </p><div style="font-style:italic"><p>Thursday, 16 October 2014 13:30</p>
</div><p style="float: left;margin: 5px;margin-right: 20px"><a href="http://uk.geekout.ee/speakers/simon-ritter/"><img src="http://uk.geekout.ee/wp-content/uploads/sites/2/2014/06/SimonRitter.jpg" alt="SimonRitter" width="150" height="190" class="alignnone size-full wp-image-451" /></a></p>
<p>Java SE 8 introduces a number of new features designed to make developers more productive.  The biggest of these is the combination of Lambda statements with the Streams API to give programmers a more functional approach to bulk operations on collections.  Other features are things like the extension of where annotations can be used to allow more errors to be found at compile time rather than run time.  This session will review these, and other features in Java SE 8 so that you can take advantage of them as soon as you start your next project.</p>
