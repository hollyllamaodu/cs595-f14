<h3 class='post-title entry-title'>
<a href='http://pyfound.blogspot.com/2014/01/support-python-software-foundation-in.html'>Support the Python Software Foundation in 2014</a>
</h3>
<div class='post-header'>
<div class='post-header-line-1'></div>
</div>
<div class='post-body entry-content'>
Happy New Year from the Python Software Foundation! 2013 was a busy year for our global community, in almost <a href="https://wiki.python.org/moin/LocalUserGroups" target="_blank">200 user groups</a>, dozens of <a href="http://www.pycon.org/" target="_blank">regional conferences</a>, and countless open source projects, sprints, online fora, and outreach events.<br />
<br />
Please help us continue to invest in and support the community in 2014 with a donation. There are 2 main ways to help:<br />
<ol>
<li><a href="http://python.org/psf/donations/" target="_blank">Donate as an individual</a></li>
<li><a href="http://www.python.org/psf/sponsorship/" target="_blank">Donate as an organization</a></li>
</ol>
The <a href="http://www.python.org/psf/" target="_blank">Python Software Foundation</a> is a 501(c)(3) non-profit. For US taxpayers, contributions to the PSF are tax-deductible. Your employer may also match your donations!<br />
<br />
<b>What will your money support?</b><br />
<br />
<table align="center" cellpadding="0" cellspacing="0" class="tr-caption-container" style="margin-left: auto; margin-right: auto; text-align: center;"><tbody>
<tr><td style="text-align: center;"><img height="266" src="https://farm8.staticflickr.com/7140/7680707056_ab7742154c.jpg" style="margin-left: auto; margin-right: auto;" width="400" /></td></tr>
<tr><td class="tr-caption" style="text-align: center;">Group debugging at a PSF-sponsored Boston Python workshop</td></tr>
</tbody></table>
The PSF sponsors conferences and community events across the globe, including in 2013 alone: SciPy, PyCon Canada, PyOhio, PyCon Ireland, PyData Boston, PythonBrasil, Kiwi PyCon, PyCon Argentina, PyConDE, RuPy, PyConUK, PyDay Ecuador, PyConZA, and the first ever PyTennessee...whew!<br />
<br />
We provide fiscal sponsorship, cover hosting costs, and sponsor workshop for user groups. We invest in the next generation of Python programmers by supporting events like <a href="http://juliaelman.com/blog/2013/07/31/durham-teen-tech-camp-press-release/" target="_blank">Teen Tech Camp</a> and <a href="http://www.toranbillups.com/blog/archive/2013/06/30/Teaching-python-to-fifth-graders/" target="_blank">Raspberry Pi programming in classrooms</a>.<br />
<br />
With support from the PSF's Outreach and Education program, <a href="http://sf.adacamp.org/" target="_blank">Ada Camp</a> came to San Francisco, scientists learned <a href="http://software-carpentry.org/" target="_blank">Software Carpentry</a>, <a href="http://litablog.org/2013/07/introduction-to-python-at-ala-annual-or-awesome-true-ala2013py/" target="_blank">librarians learned Python</a> at the ALA Annual Conference, and introductory workshops were run by user groups around the world.<br />
<br />
The PSF <a href="http://www.python.org/psf/grants/" target="_blank">Grants program</a> enables experimental development on projects like PyPy, and the <a href="http://pythonsprints.com/" target="_blank">Sprints program</a> supports development on your favorite open source Python projects. We enforce Python's trademarks, protect Python's intellectual property, keep PyPI running, and are bringing you the <a href="http://preview.python.org/" target="_blank">next version of python.org</a>.<br />
<br />
Help us do all this and more in 2014 and <a href="http://python.org/psf/donations/">donate today</a>.

We invest in the next generation of Python programmers by supporting events like <a href="http://juliaelman.com/blog/2013/07/31/durham-teen-tech-camp-press-release/" target="_blank">Teen Tech Camp</a> and <a href="http://www.toranbillups.com/blog/archive/2013/06/30/Teaching-python-to-fifth-graders/" target="_blank">Raspberry Pi programming in classrooms</a>.<br />
<br />
With support from the PSF's Outreach and Education program, <a href="http://sf.adacamp.org/" target="_blank">Ada Camp</a> came to San Francisco, scientists learned <a href="http://software-carpentry.org/" target="_blank">Software Carpentry</a>, <a href="http://litablog.org/2013/07/introduction-to-python-at-ala-annual-or-awesome-true-ala2013py/" target="_blank">librarians learned Python</a> at the ALA Annual Conference, and introductory workshops were run by user groups around the world.<br />
<br />
The PSF <a href="http://www.python.org/psf/grants/" target="_blank">Grants program</a> enables experimental development on projects like PyPy, and the <a href="http://pythonsprints.com/" target="_blank">Sprints program</a> supports development on your favorite open source Python projects. We enforce Python's trademarks, protect Python's intellectual property, keep PyPI running, and are bringing you the <a href="http://preview.python.org/" target="_blank">next version of python.org</a>.<br />
<br />
Help us do all this and more in 2014 and <a href="http://python.org/psf/donations/">donate today</a>.</div>
