	<h3 class="entry-title">
			Java's New Console Tool
	    </h3>

    <h4 class="entry-meta">By Tori Wieldt on <a href="#">Sep 18, 2014</a></h4>

    <div class="entry-body">
                                        	<p>Included in Java SE 8 update 20 is a new feature: the <strong><a href="http://www.oracle.com/technetwork/java/javaseproducts/advanced-mgmt/advancedmanagementconsole-2254207.html">Java Advanced Management Console</a>.&nbsp;</strong>&quot;We think any sys admin dealing with a Java environment will find this tool extremely helpful,&quot; said Peter Utzschneider, VP of Product Management at Oracle.</p> 
  <p><strong><a href="http://www.oracle.com/technetwork/java/javaseproducts/advanced-mgmt/advancedmanagementconsole-2254207.html">Java Advanced Management Console</a></strong>&nbsp;provides you ways to monitor and understand the client systems you manage. It allows organizations to track usage and more easily create and manage client configuration like Deployment Rule Sets (DRS). DRS can control execution of tracked applications as well as specify compatibility of which application should use which Java SE installation.</p> 
  <p><a href="http://www.oracle.com/technetwork/java/javaseproducts/advanced-mgmt/advancedmanagementconsole-2254207.html"><img src="https://blogs.oracle.com/java-platform-group/resource/NewManagementConsoleinJavaSEAdvanced8u20/usageTracking.png" hspace="10" vspace="10" /></a></p> 
  <p><a href="http://www.oracle.com/technetwork/java/javaseproducts/advanced-mgmt/advancedmanagementconsole-2254207.html"></a>The Advanced Management Console is part of Java SE Advanced designed for desktop administrators, whose users need to run many different Java applications. It provides&nbsp;<a href="http://docs.oracle.com/javacomponents/usage-tracker/overview/index.html" target="_blank">usage tracking</a> for those Applet &amp; Web Start applications to help identify them for guided DRS creation. DRS can then be verified against the tracked data, to ensure that end-users can run their application against the appropriate Java version with no prompts.</p> 
  <p>Usage tracking also has a different definition for Java SE than it does for most software applications. Unlike most applications where usage can be determined by a simple run-count, Java is a platform used for launching other applications. This means that usage tracking must answer both &quot;how often is this Java SE version used&quot; and &quot;what applications are launched by it.&quot;&nbsp;</p> 
  <p>In this video, Jim Weaver, Java Technology Ambassador at Oracle, walks through a demonstration of the new Java Advanced Management Console (AMC) tool.
  </p> 
  <p> 
    <iframe width="560" height="315" src="https://www.youtube.com/embed/5OLGAsMWWsI" frameborder="0"></iframe> 
  </p> 
  <p><strong><a href="http://www.oracle.com/technetwork/java/javaseproducts/advanced-mgmt/advancedmanagementconsole-2254207.html">Java Advanced Management Console</a></strong>&nbsp;is one of the features in Java SE Advanced. It may be <a href="http://www.oracle.com/technetwork/java/javase/downloads/amc-download-2255283.html">downloaded</a> for development and testing according to the terms set forth in the Oracle Binary Code License.</p>
